# LLM Final Project Code

Code for the CAS4133 Large Language Models final project  
Yonsei University, Spring 2026

**Chan Joo Jung**

## Project Title

**When Do LLMs Say "I Don't Know"?: Diagnosing and Repairing the Gap Between Verbalized Confidence and Real Uncertainty**

## Overview

This repository contains the code used for the final project investigating the relationship between verbalized confidence and actual uncertainty in large language models (LLMs). The project analyzes confidence calibration, uncertainty estimation through sampling consistency, and proposes a Consistency-Calibrated Abstention (CCA) framework for improving confidence reliability and reducing hallucinations on unanswerable questions.
